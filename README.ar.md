## 📖 نظرة عامة

**APIScan Ai** هو تطبيق مكتبي فائق الأداء صُمم خصيصاً للمطورين، والباحثين، والمحترفين في مجال الذكاء الاصطناعي. يتيح لك التطبيق فحص وتدقيق قواعد بيانات ضخمة من مفاتيح واجهات برمجة التطبيقات (API Keys) لأكبر منصات الذكاء الاصطناعي (مثل OpenAI, Anthropic, Gemini, Groq والمزيد) في وقت قياسي.

تم بناء التطبيق باستخدام نواة صلبة من لغة **Rust** وواجهة مستخدم عصرية بـ **React** عبر إطار عمل **Tauri**، مما يقضي على مشاكل التقطيع وانهيار المتصفح المعتادة في أدوات الفحص التي تعتمد على الويب. يمكن للبرنامج التعامل بسهولة مع ملفات تتجاوز **50,000 مفتاح** (أكثر من 15 ميجابايت) بثبات تام.

---

## ✨ الميزات الجوهرية

- 🚀 **محرك كسر القيود (Limit-Breaker Engine)**: خوارزمية مخصصة تقرأ ملفات الـ CSV و XLSX العملاقة بنظام التدفق (Streaming). تقرأ البيانات من القرص الصلب مباشرة وتفحصها دون تكديسها في الذاكرة العشوائية (RAM).
- 🧠 **التعرف الذكي التلقائي**: نظام ذكي يكتشف نوع مزود الخدمة (AI Provider) بمجرد النظر إلى صيغة المفتاح، دون الحاجة لأي تدخل أو اختيار يدوي منك.
- 👑 **اكتشاف نماذج النخبة (Flagship Models)**: لا يكتفي البرنامج بمعرفة ما إذا كان المفتاح يعمل أم لا، بل لديه القدرة على فحص ما إذا كان المفتاح يمتلك صلاحيات وصول للنماذج المدفوعة الرائدة (مثل `GPT-4o` و `Claude 3.5 Sonnet`).
- 🔒 **أمان وخصوصية 100%**: التطبيق يعمل بشكل محلي بالكامل. يتم إرسال طلبات الفحص من جهازك والـ IP الخاص بك مباشرة إلى سيرفرات شركات الذكاء الاصطناعي. لا توجد أي خوادم وسيطة تحفظ أو تطلع على مفاتيحك. أصولك الرقمية في أمان تام.
- 🌍 **دعم كامل للغة العربية**: واجهة مصممة لتدعم اللغتين العربية والإنجليزية بشكل ديناميكي مع محاذاة صحيحة (RTL/LTR).
- 💎 **تصميم عصري (Glassmorphism)**: واجهة مستخدم فائقة الجمال والسرعة، تعتمد على طابع الزجاج المعتم مع تأثيرات حركية دقيقة باستخدام Framer Motion و Tailwind CSS.

## 🏗️ البنية التقنية (Architecture)

### النواة الخلفية (Rust / Tauri)
- **معالجة البيانات**: استخدام مكتبة `csv` للتعامل المرن مع الأعمدة غير المتطابقة، ومكتبات `quick-xml` و `zip` للتعامل المباشر مع ملفات إكسل المُعقدة بمختلف ظروفها.
- **التزامن (Concurrency)**: استخدام الـ `tokio` و `Semaphore` لعمل طلبات شبكة متوازية سريعة جداً دون تجاوز حدود نظام التشغيل.

### الواجهة الأمامية (React 19 / TypeScript)
- **واجهة المستخدم**: مبنية بأحدث إصدار من React مع ضمان استقرار الأنواع باستخدام TypeScript.
- **النظام البصري**: Tailwind V4 لتخصيص الواجهة بسرعة مع تأثيرات بصرية صُممت خصيصاً للبرنامج.
- **إدارة الحالة**: تواصل لحظي مستمر بين نواة Rust والواجهة الأمامية لعرض الإحصائيات الحية وسجلات الفحص.

## 🚀 طريقة التشغيل وتثبيت التطبيق

### 1. المتطلبات الأساسية (Prerequisites)
تأكد من تنصيب البرامج التالية على جهازك:
- [Node.js](https://nodejs.org/en/) (إصدار 18 أو أحدث)
- [Rust](https://www.rust-lang.org/tools/install) 
- أدوات بناء C++ (لمستخدمي ويندوز: Visual Studio C++ Build tools)

### 2. استنساخ الكود (Clone)
```bash
git clone https://github.com/Omar-Tnzxo/ApiScan-Ai.git
cd ApiScan-Ai
```

### 3. تثبيت الحزم
```bash
npm install
```

### 4. التشغيل في وضع التطوير (Development)
```bash
npm run tauri dev
```
*(ملاحظة: التشغيل للمرة الأولى قد يأخذ بضع دقائق ريثما يقوم الـ Compiler بتجهيز حزم لغة Rust).*

---

## 📦 استخراج النسخة النهائية (Production Build)

لتحويل الكود إلى ملف `.exe` تنفيذي جاهز للاستخدام المباشر على ويندوز:

```bash
npm run tauri build
```

بعد انتهاء العملية بنجاح، ستجد ملف التثبيت والملف التنفيذي في المسار التالي:
`src-tauri/target/release/bundle/nsis/`
